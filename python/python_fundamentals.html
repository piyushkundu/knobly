A Comprehensive Guide to Python Programming

Introduction to Python

Python is a high-level, interpreted programming language known for its readability and simplicity. Created by Guido van Rossum and first released in 1991, Python's design philosophy emphasizes code readability with its notable use of significant whitespace.

Development of Python

Guido van Rossum began working on Python in the late 1980s as a successor to the ABC programming language. He wanted to create a language that was extensible and had better exception handling than its predecessors. The name "Python" was inspired by the British comedy group Monty Python, as van Rossum was a fan of Monty Python's Flying Circus.

Python was designed to be a highly readable language. Its formatting is visually uncluttered, and it frequently uses English keywords where other languages use punctuation. Python's syntax allows programmers to express concepts in fewer lines of code than possible in languages such as C++ or Java.

Advantages of Python

Easy to Learn and Use

Python has a simple, easy-to-learn syntax that emphasizes readability, reducing program maintenance costs and making it accessible to beginners.

Versatility

Python is used in various domains: web development, data analysis, AI, scientific computing, automation, and more.

Large Standard Library

Python comes with a vast standard library that provides modules for various tasks, reducing the need for external libraries.

Interpreted Language

Being an interpreted language, Python code is executed line by line, making debugging easier and more efficient.

Open Source

Python is free to use and distribute, even for commercial purposes, under an OSI-approved open-source license.

Strong Community Support

Python has a large and active community that contributes to libraries, provides support, and shares knowledge.

These advantages make Python an excellent choice for both beginners and experienced developers. Its flexibility and ease of use have contributed significantly to its growing popularity in recent years.

Working with Python

Installing Python

Before you can start programming in Python, you need to download and install it on your computer. Here's how:

  1. Download Python: Visit the official Python website at python.org/downloads and download the latest version for your operating system.

  2. Run the Installer: Double-click the downloaded file to run the installer.

    • On Windows: Make sure to check the box that says "Add Python to PATH" before clicking Install Now.
    • On macOS: Follow the instructions in the installer.
    • On Linux: Most distributions come with Python pre-installed. If not, use your package manager (e.g., apt-get install python3 for Debian/Ubuntu).

  3. Verify Installation: Open a command prompt or terminal and type:

    python --version

    You should see the Python version number displayed.

Python Interactive Mode

Python offers an interactive mode where you can write and execute code line by line. This is great for testing small code snippets and learning Python.

To start the interactive mode:

  1. Open a command prompt or terminal
  2. Type python or python3 and press Enter
  3. You should see the Python prompt >>>
>>> print("Hello, World!")
Hello, World!
>>> 2 + 3
5
>>> name = "Python"
>>> print(f"I love {name} programming!")
I love Python programming!

To exit the interactive mode, type exit() or press Ctrl+Z on Windows or Ctrl+D on Unix-like systems.

Python Program Mode

For larger programs, you'll want to write your code in a file and execute it. This is called program mode or script mode.

  1. Create a new text file with a .py extension (e.g., program.py)
  2. Write your Python code in the file
  3. Save the file
  4. Run the program from the command line with: python program.py
# This is program.py
print("Running Python in program mode")

for i in range(5):
    print(f"Count: {i}")

When you run this program with python program.py, it will execute all the code at once and display the output.

Python Tokens

In Python, a token is the smallest unit of a program. The Python interpreter breaks down a Python program into tokens before parsing it. There are five types of tokens in Python:

Identifiers

An identifier is a name used to identify a variable, function, class, module, or other object. Python identifiers follow these rules:

  • An identifier can only start with a letter (A-Z, a-z) or underscore (_)
  • The rest of the identifier can consist of letters, underscores, and digits (0-9)
  • Identifiers are case-sensitive (name, Name, and NAME are three different identifiers)
  • Reserved keywords cannot be used as identifiers
  • Special symbols like !, @, #, $, % cannot be used in identifiers
  • There is no limit on the length of an identifier

Valid identifier examples:

name
student_1
_private
totalSum
get_value
UserAccount

Invalid identifier examples:

123name  # Cannot start with a digit
my-var   # Hyphen not allowed
for      # Reserved keyword
$amount  # Special symbol not allowed

Literals

Literals are data items that have a fixed value. Python supports different types of literals:

String Literals

String literals are enclosed within single quotes (' '), double quotes (" "), or triple quotes (''' ''' or """ """).

Single-line String:

name = 'John'
message = "Hello, World!"

Multi-line String using Triple Quotes:

long_text = """This is a long text
that spans over multiple lines.
Triple quotes allow this."""

Multi-line String using Backslash:

another_text = "This is another long text \
that continues on the next line \
using backslashes."

Numeric Literals

Numeric literals include integers, floating-point numbers, and complex numbers.

Integer Literals:

a = 10      # Decimal
b = 0b1010  # Binary (starts with 0b)
c = 0o12    # Octal (starts with 0o)
d = 0xA     # Hexadecimal (starts with 0x)

Float Literals:

x = 10.5
y = 1.5e2    # Scientific notation (150.0)
z = .5       # Same as 0.5

Complex Literals:

Complex numbers are written in the form of a + bj, where a is the real part and b is the imaginary part.

complex_num = 3 + 5j

Boolean Literals

Boolean literals represent truth values. There are only two Boolean literals: True and False.

is_active = True
is_completed = False

None Literal

The None literal represents the absence of a value or a null value.

result = None

It's often used as a placeholder for optional or missing values. It evaluates to False in boolean contexts.

Identity Operators

Identity operators are used to compare objects, not if they are equal, but if they are actually the same object with the same memory location.

Operator Description Example
is Returns True if both variables are the same object x is y
is not Returns True if both variables are not the same object x is not y 
a = [1, 2, 3]
b = [1, 2, 3]
c = a

print(a is b)    # False (different objects with the same value)
print(a is c)    # True (same object)

Membership Operators

Membership operators are used to test if a sequence is presented in an object.

Operator Description Example
in Returns True if a sequence with the specified value is present in the object x in y
not in Returns True if a sequence with the specified value is not present in the object x not in y 
fruits = ["apple", "banana", "cherry"]
print("banana" in fruits)     # True
print("pear" not in fruits)   # True

Python Punctuation

Python uses various punctuation marks for different purposes:

  • Colon (:) - Used in statements like if, for, while, class, and function definitions.
  • Semicolon (;) - Can be used to separate multiple statements on a single line.
  • Comma (,) - Used to separate items in lists, tuples, and function arguments.
  • Period (.) - Used for attributes and methods access or for floating-point numbers.
  • Parentheses () - Used in function calls, expressions, and tuples.
  • Square brackets [] - Used for lists and indexing.
  • Curly braces {} - Used for dictionaries and sets.
  • Hash (#) - Used for comments.
# Example of Python punctuation usage
def greet(name):  # Function definition with colon
    print("Hello, " + name + "!")  # Period in function name
    
names = ["Alice", "Bob", "Charlie"]  # Square brackets for list, commas separate items
person = {"name": "David", "age": 30}  # Curly braces for dictionary
                
result = (1 + 2) * 3  # Parentheses affect operator precedence

Print Statement in Python

The print() function is used to output data to the standard output device (screen).

Basic Print Usage

print("Hello, World!")  # Outputs: Hello, World!
print(10)             # Outputs: 10
print(True)           # Outputs: True

Printing Multiple Items

print("Name:", "Alice", "Age:", 25)  # Outputs: Name: Alice Age: 25

Print with Separator and End Parameters

# Custom separator
print("apple", "banana", "cherry", sep="|")  # Outputs: apple|banana|cherry

# Custom end character (default is newline)
print("Hello", end=" ")
print("World")  # Outputs: Hello World

Formatted String Literals (f-strings)

name = "Alice"
age = 25
print(f"My name is {name} and I am {age} years old.")
# Outputs: My name is Alice and I am 25 years old.

Format Method

print("My name is {} and I am {} years old.".format(name, age))
# Outputs: My name is Alice and I am 25 years old.

# With position indexes
print("My name is {0} and I am {1} years old.".format(name, age))

# With named parameters
print("My name is {n} and I am {a} years old.".format(n=name, a=age))

Old-style String Formatting

print("My name is %s and I am %d years old." % (name, age))
# Outputs: My name is Alice and I am 25 years old.

Input Statement in Python

The input() function allows user input from the standard input device (keyboard).

Basic Input Usage

name = input("Enter your name: ")
print("Hello, " + name + "!")

Note: The input() function always returns a string, even if the user enters a number.

Converting Input to Other Types

# Converting input to an integer
age = int(input("Enter your age: "))
next_year_age = age + 1
print(f"Next year, you will be {next_year_age} years old.")

# Converting input to a float
height = float(input("Enter your height in meters: "))
print(f"Your height is {height} meters.")

Input Validation Example

while True:
    try:
        number = int(input("Enter a positive number: "))
        if number <= 0:
            print("That's not a positive number. Try again.")
        else:
            break
    except ValueError:
        print("That's not a valid number. Try again.")
        
print(f"You entered: {number}")

Multiple Inputs

# Getting multiple inputs on separate lines
first_name = input("Enter your first name: ")
last_name = input("Enter your last name: ")
print(f"Hello, {first_name} {last_name}!")

# Getting multiple inputs on a single line
x, y = input("Enter two numbers separated by space: ").split()
x, y = int(x), int(y)
print(f"Sum: {x + y}")
