Introduction to CSS

What is CSS?

CSS (Cascading Style Sheets) is a style sheet language used to describe the presentation of a web page. It allows developers to control the layout, color, font, and other visual aspects of a web page.

CSS works in harmony with HTML, which provides the structure and content of a web page, while CSS handles the styling and visual design.

History of CSS

CSS was first proposed in 1994 by HÃ¥kon Wium Lie, who was working at the European Organization for Nuclear Research (CERN) at the time. The goal was to create a standard way to style web pages, as the early web pages were primarily text-based and lacked visual appeal.

The first CSS specification, CSS1, was published in 1996 by the World Wide Web Consortium (W3C). Over the years, CSS has evolved through several versions, with CSS2 being released in 1998 and CSS3 in 1999, which introduced many new features and capabilities.

Characteristics of CSS

  • Time-saving: CSS allows you to apply styles to multiple HTML elements at once, saving time and effort compared to inline styles.
  • Superior to HTML: CSS provides more advanced styling options, such as layout control, typography, and animations, which are not easily achievable with HTML alone.
  • Offline browsing: CSS files can be cached by the browser, allowing for faster page loading and offline browsing.
  • Separation of concerns: CSS separates the presentation from the content, making it easier to maintain and update the website's appearance.

Types of CSS

There are three main ways to apply CSS to an HTML document:

Inline CSS

<p style="color: blue; font-size: 16px;">This is a paragraph with inline CSS.</p>

Inline CSS is applied directly to an HTML element using the style attribute.

Internal CSS

<style>
p { color: blue; font-size: 16px; }
</style>

Internal CSS is placed within the <head> section of the HTML document.

External CSS

<link rel="stylesheet" href="styles.css">

External CSS is applied by linking to a CSS file in the HTML document.

Priorities of CSS

When multiple CSS rules are applied to the same element, the browser determines which rule takes precedence based on the following order of priority:

  1. Inline Styles: Inline CSS, applied directly to an element using the style attribute, has the highest priority. It overrides all other styles.
    2. <p style="color: red;">This is red text.</p>
  2. Internal Styles: Internal CSS rules, defined within the <style> tag in the <head> section of the HTML document, take precedence over external CSS.
    3. <style>
p { color: green; }
</style>
  3. External Styles: External CSS, linked through a separate stylesheet file using the <link> tag, is overridden by both inline and internal styles.
    4. <link rel="stylesheet" href="styles.css">
  4. Browser Default Styles: If no CSS is applied, the browser's default styles are used. These have the lowest priority.

Additionally, specificity and importance further influence CSS priorities:

  • Specificity: A more specific selector (e.g., an ID) overrides a less specific one (e.g., a class or element selector).
  • !important: The !important declaration forces a CSS rule to override all others, regardless of specificity or source.
    • p {
  color: blue !important;
}
