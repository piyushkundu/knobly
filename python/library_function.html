Introduction to Python Libraries

Python's standard library is a collection of modules that provide access to system functionality such as file I/O as well as modules that are written in Python that provide standardized solutions for many problems that occur in everyday programming.

The Python Standard Library is distributed with Python and provides a rich set of modules that extend the functionality of Python. These modules make Python a powerful language for various tasks from mathematical computations to web development.

In this guide, we'll explore some of the most commonly used modules:

  • Math Module - For mathematical operations and functions
  • Random Module - For generating random numbers
  • String Module - For string manipulation operations
  • Date and Time Modules - For handling dates and times

Importing Modules in Python

Before you can use a module, you need to import it. Python provides several ways to import modules:

Standard Import

Import the entire module. You'll need to use the module name as a prefix when accessing functions.

import math
result = math.sqrt(16)  # Returns 4.0
From Import

Import specific functions from a module, eliminating the need for the module prefix.

from math import sqrt
result = sqrt(16)  # Returns 4.0
Import with Alias

Import a module with a different name (alias) to make it shorter or avoid name conflicts.

import math as m
result = m.sqrt(16)  # Returns 4.0
Import All (Not Recommended)

Import all functions from a module. This is generally not recommended as it can lead to name conflicts.

from math import *
result = sqrt(16)  # Returns 4.0

Math Module

The math module provides mathematical functions defined by the C standard. These functions cannot be used with complex numbers; the cmath module provides similar functionality for complex numbers.

abs()

Returns the absolute value of a number.

import math
math.abs(-5)  # Returns 5

Note: Python has a built-in abs() function that works similarly.

sqrt()

Returns the square root of a number.

import math
math.sqrt(16)  # Returns 4.0
math.sqrt(2)   # Returns 1.4142135623730951
exp()

Returns e raised to the power of x.

import math
math.exp(1)   # Returns 2.718281828459045 (e)
math.exp(2)   # Returns 7.3890560989306495 (e¬≤)
ceil()

Returns the ceiling of x, the smallest integer greater than or equal to x.

import math
math.ceil(4.1)  # Returns 5
math.ceil(4.0)  # Returns 4
floor()

Returns the floor of x, the largest integer less than or equal to x.

import math
math.floor(4.9)  # Returns 4
math.floor(4.0)  # Returns 4
log()

Returns the natural logarithm of x (base e).

import math
math.log(math.e)  # Returns 1.0
math.log(100, 10) # Returns 2.0 (log base 10)
log10()

Returns the base-10 logarithm of x.

import math
math.log10(100)  # Returns 2.0
math.log10(1000) # Returns 3.0
pow()

Returns x raised to the power y.

import math
math.pow(2, 3)  # Returns 8.0
math.pow(9, 0.5) # Returns 3.0 (square root of 9)
sin()

Returns the sine of x (x in radians).

import math
math.sin(0)        # Returns 0.0
math.sin(math.pi/2) # Returns 1.0
cos()

Returns the cosine of x (x in radians).

import math
math.cos(0)        # Returns 1.0
math.cos(math.pi) # Returns -1.0
tan()

Returns the tangent of x (x in radians).

import math
math.tan(0)   # Returns 0.0
math.tan(math.pi/4) # Returns 0.9999999999999999 (approximately 1)
degrees()

Converts an angle from radians to degrees.

import math
math.degrees(math.pi)      # Returns 180.0
math.degrees(math.pi/2)   # Returns 90.0
radians()

Converts an angle from degrees to radians.

import math
math.radians(180)  # Returns 3.141592653589793 (œÄ)
math.radians(90)   # Returns 1.5707963267948966 (œÄ/2)
trunc()

Returns the truncated integer value of x (towards zero).

import math
math.trunc(4.9)  # Returns 4
math.trunc(-4.9) # Returns -4

Random Module

The random module implements pseudo-random number generators for various distributions. For cryptographic purposes, use the secrets module instead.

random()

Returns a random floating point number in the range [0.0, 1.0).

import random
random.random()  # Returns a random float between 0.0 and 1.0
# Example: 0.7134566380732742
randint()

Returns a random integer N such that a <= N <= b.

import random
random.randint(1, 10)  # Returns a random integer between 1 and 10 (inclusive)
# Example: 7
choice()

Returns a random element from a non-empty sequence.

import random
random.choice(['apple', 'banana', 'cherry'])  # Might return 'banana'
shuffle()

Shuffles a sequence in place.

import random
deck = ['ace', 'king', 'queen', 'jack']
random.shuffle(deck)
print(deck)  # Might print ['king', 'ace', 'jack', 'queen']

String Manipulation

Python provides various ways to manipulate strings, both through operators and methods.

String Operators

Concatenation (+)

Joins two or more strings.

first_name = "John"
last_name = "Doe"
full_name = first_name + " " + last_name  # "John Doe"
Repetition (*)

Repeats a string a specified number of times.

dash = "-"
line = dash * 20  # "--------------------"
Membership (in)

Checks if a substring exists within a string.

text = "Python is amazing"
"Python" in text  # Returns True
"Java" in text    # Returns False

String Slicing

Basic Slicing

Extract a portion of a string using indices.

text = "Python Programming"
text[0]     # Returns "P" (first character)
text[7:18]  # Returns "Programming" (from index 7 to 17)
text[:6]    # Returns "Python" (from start to index 5)
text[7:]    # Returns "Programming" (from index 7 to end)
Step Slicing

Extract with a specific step increment.

text = "Python Programming"
text[::2]   # Returns "Pto rgamn" (every 2nd character)
text[::-1]  # Returns "gnimmargorP nohtyP" (reversed)

String Methods

upper(), lower(), title(), capitalize()

Change the case of a string.

text = "python programming"
text.upper()       # Returns "PYTHON PROGRAMMING"
text.title()       # Returns "Python Programming"
text.capitalize()  # Returns "Python programming"
strip(), lstrip(), rstrip()

Remove whitespace or specified characters from a string.

text = "  Hello, World!  "
text.strip()    # Returns "Hello, World!"
text.lstrip()   # Returns "Hello, World!  "
text.rstrip()   # Returns "  Hello, World!"
replace()

Replace occurrences of a substring.

text = "Hello, World!"
text.replace("World", "Python")  # Returns "Hello, Python!"
split(), join()

Split a string into a list or join a list into a string.

text = "apple,banana,cherry"
fruits = text.split(",")  # Returns ["apple", "banana", "cherry"]

fruits = ["apple", "banana", "cherry"]
text = ", ".join(fruits)  # Returns "apple, banana, cherry"
find(), index()

Find the position of a substring.

text = "Python is amazing"
text.find("is")     # Returns 7
text.find("Java")   # Returns -1 (not found)
text.index("is")    # Returns 7
# text.index("Java") would raise a ValueError

Date and Time Functions

Python provides several modules for handling dates and times, including the time and calendar modules.

Time Module

time()

Returns the current time in seconds since the Epoch (January 1, 1970).

import time
current = time.time()  # Returns something like 1647354861.7639298
ctime()

Converts a time expressed in seconds since the Epoch to a string.

import time
time.ctime()  # Returns current time as string like 'Tue Mar 15 13:41:01 2022'
time.ctime(0)  # Returns 'Thu Jan  1 00:00:00 1970'
sleep()

Suspends execution for the given number of seconds.

import time
print("Hello")
time.sleep(2)  # Pauses for 2 seconds
print("World")
strftime()

Formats time according to the rules of the specified format.

import time
time.strftime("%Y-%m-%d %H:%M:%S")  # Example: '2022-03-15 13:45:30'

Calendar Module

month()

Returns a month's calendar as a multi-line string.

import calendar
print(calendar.month(2022, 3))  # Prints March 2022 calendar
isleap()

Returns True if the specified year is a leap year.

import calendar
calendar.isleap(2020)  # Returns True
calendar.isleap(2022)  # Returns False
weekday()

Returns the day of the week (0 is Monday, 6 is Sunday) for the given date.

import calendar
# What day of the week was January 1, 2022?
calendar.weekday(2022, 1, 1)  # Returns 5 (Saturday)

Datetime Module

datetime.now()

Returns the current local date and time.

from datetime import datetime
now = datetime.now()  # Current date and time
print(now)  # Example: 2022-03-15 13:50:25.123456
datetime.strftime()

Formats datetime object as a string.

from datetime import datetime
now = datetime.now()
formatted = now.strftime("%Y-%m-%d %H:%M:%S")
print(formatted)  # Example: '2022-03-15 13:50:25'
datetime.strptime()

Parses a string representing a time according to a format.

from datetime import datetime
date_string = "15/03/2022"
date_object = datetime.strptime(date_string, "%d/%m/%Y")
print(date_object)  # 2022-03-15 00:00:00
timedelta

Represents a duration, the difference between two dates or times.

from datetime import datetime, timedelta
now = datetime.now()
# Add 10 days to current date
future = now + timedelta(days=10)
print(future)  # Example: 2022-03-25 13:55:30.123456

Random Module Examples

Generate a Random Sample
import random

# Sample k unique elements from a population
population = range(1, 51)  # Numbers from 1 to 50
sample = random.sample(population, 6)
print(sample)  # Example: [7, 13, 47, 21, 38, 5]

# Generate random float in a range
ranged_float = random.uniform(1.0, 10.0)
print(ranged_float)  # Example: 4.527834982741

String Manipulation Examples

Advanced String Formatting
# Using f-strings (Python 3.6+)
name = "Alice"
age = 30
message = f"Hello, {name}! You are {age} years old."
print(message)  # Output: Hello, Alice! You are 30 years old.

# Using format() method
template = "Hello, {}! You are {} years old."
message = template.format(name, age)
print(message)  # Output: Hello, Alice! You are 30 years old.

# Using named placeholders
template = "Hello, {name}! You are {age} years old."
message = template.format(name=name, age=age)
print(message)  # Output: Hello, Alice! You are 30 years old.

Date and Time Examples

Date Arithmetic
from datetime import datetime, timedelta

# Current date and time
now = datetime.now()
print(f"Current date and time: {now}")

# Add 7 days to current date
one_week_later = now + timedelta(days=7)
print(f"One week later: {one_week_later}")

# Subtract 12 hours
twelve_hours_ago = now - timedelta(hours=12)
print(f"12 hours ago: {twelve_hours_ago}")

# Calculate difference between dates
future_date = datetime(2025, 12, 31)
time_remaining = future_date - now
print(f"Days until end of 2025: {time_remaining.days}")
