Python File Handling

File handling is an essential part of any programming language, and Python makes it simple with its intuitive interface. This guide provides a comprehensive overview of file operations in Python, ranging from basic text file handling to advanced binary and object-oriented file processing.

Why File Handling is Important

  • Persistent data storage beyond program execution
  • Processing large datasets that can't fit in memory
  • Data exchange between applications
  • Configuration management
  • Logging and record keeping

Python supports various file types including:

  • Text files: Human-readable files containing text data
  • Binary files: Files containing binary data, such as images, audio, and custom data structures
  • CSV files: Comma-separated values for tabular data
  • JSON files: JavaScript Object Notation for structured data
  • XML files: Extensible Markup Language for hierarchical data

Let's dive into each type of file processing to understand how Python handles different file operations.

10.2 Text File Processing

Text files store data in human-readable format, with each character represented by its ASCII or Unicode value. Python provides simple and efficient ways to read from and write to text files.

10.2.1 open()

Before you can read or write a file, you need to open it. The built-in open() function returns a file object that can be used to interact with the file.

Syntax 
file_object = open(filename, mode)

The open() function takes two parameters:

  • filename: The path to the file you want to open.
  • mode: A string that specifies the purpose of opening the file.

Common File Modes:

  • 'r': Read (default) - Opens a file for reading
  • 'w': Write - Opens a file for writing. Creates a new file if it doesn't exist or truncates the file if it exists
  • 'a': Append - Opens a file for appending. Creates a new file if it doesn't exist
  • 'x': Exclusive creation - Creates a new file, fails if the file already exists
  • 't': Text mode (default)
  • 'b': Binary mode
  • '+': Update (read and write)
Example 
# Opening a file for reading
f = open('example.txt', 'r')

# Opening a file for writing
f = open('example.txt', 'w')

# Opening a file for appending
f = open('example.txt', 'a')

# Opening a binary file for reading
f = open('image.jpg', 'rb')

Note: It's a good practice to use relative or absolute paths when opening files, especially if your file is not in the same directory as your Python script.

10.2.2 close()

When you're done with a file, it's important to close it using the close() method. This frees up system resources and ensures that all data is written to the file.

Syntax 
file_object.close()
Example 
# Open a file
f = open('example.txt', 'w')

# Do some operations with the file
f.write('Hello, World!')

# Close the file
f.close()

Best Practice: Use the with statement, which automatically closes the file after the block is executed, even if an exception occurs:

Using with statement 
with open('example.txt', 'w') as f:
    f.write('Hello, World!')
# File is automatically closed after the block

10.2.3 write()

The write() method writes a string to the file. It returns the number of characters written.

Syntax 
file_object.write(string)
Example 
# Writing a single line
with open('example.txt', 'w') as f:
    f.write('Hello, World!')

# Writing multiple lines
with open('example.txt', 'w') as f:
    f.write('First line\n')
    f.write('Second line\n')
    f.write('Third line\n')

Note: The write() method does not add a newline character automatically. You need to add \n explicitly if you want a new line.

10.2.4 readline()

The readline() method reads a single line from the file. A newline character (\n) is kept at the end of the string.

Syntax 
line = file_object.readline()
Example 
# Reading a single line
                            with open('example.txt', 'r') as f:
                                line = f.readline()
                                print(line)
                            
                            # Reading all lines one by one
                            with open('example.txt', 'r') as f:
                                line = f.readline()
                                while line:
                                    print(line, end='')
                                    line = f.readline()

Warning: readline() returns an empty string when it reaches the end of the file.

10.2.5 More commands to write and read the file

Python provides several other methods to read and write files efficiently.

Reading Multiple Lines:

readlines() Method 
# Reading all lines into a list
                            with open('example.txt', 'r') as f:
                                lines = f.readlines()
                                
                            for line in lines:
                                print(line, end='')
read() Method 
# Reading the entire file content
                            with open('example.txt', 'r') as f:
                                content = f.read()
                                print(content)
File Iteration 
# Reading file line by line using a for loop
                            with open('example.txt', 'r') as f:
                                for line in f:
                                    print(line, end='')

Writing Multiple Lines:

writelines() Method 
# Writing a list of strings to a file
                            lines = ['First line\n', 'Second line\n', 'Third line\n']
                            with open('example.txt', 'w') as f:
                                f.writelines(lines)

Note: The writelines() method does not add newline characters between the strings. If you want newlines, you need to include them in your strings.

10.2.6 Random Reading in file

Python allows you to move the file pointer to any position in the file for random access.

The seek() method sets the file's current position at the specified offset.

Syntax 
file_object.seek(offset, whence)

Parameters:

  • offset: Number of bytes to be moved
  • whence: Position from where the bytes are to be moved
    • 0 - Beginning of the file (default)
    • 1 - Current position
    • 2 - End of the file
Example 
# Move to the 5th byte in the file
                            with open('example.txt', 'r') as f:
                                f.seek(5)
                                print(f.read(10))  # Read 10 bytes from position 5
                                
                            # Move to the 3rd byte before the end
                            with open('example.txt', 'r') as f:
                                f.seek(-3, 2)
                                print(f.read())  # Read from 3 bytes before the end to the end

The tell() method returns the current position of the file pointer.

Syntax 
position = file_object.tell()
Example 
with open('example.txt', 'r') as f:
                                print(f.tell())  # Prints 0 (start of file)
                                
                                f.read(5)  # Read 5 bytes
                                print(f.tell())  # Prints 5
                                
                                f.read(10)  # Read 10 more bytes
                                print(f.tell())  # Prints 15

10.3 Binary File Processing

Binary files store data in the form of bytes, making them suitable for non-text data like images, audio files, and serialized objects. Unlike text files, binary files are not human-readable and require special methods to handle them.

10.3.1 'pickle' module

The pickle module implements a powerful algorithm for serializing and de-serializing Python object structures. "Pickling" is the process of converting a Python object into a byte stream, and "unpickling" is the reverse process.

Importing pickle 
import pickle

Key Features of pickle:

  • Preserves the relationships between Python objects
  • Can serialize nearly all Python objects
  • Supports multiple protocols for backward compatibility
  • Secure against erroneous or maliciously constructed data

Security Warning: Never unpickle data received from untrusted sources, as it can execute arbitrary code during unpickling.

10.3.2 open()

To work with binary files, you need to open them in binary mode by adding 'b' to the mode string.

Syntax 
file_object = open(filename, mode)
Example 
# Open a binary file for writing
                            binary_file = open('data.bin', 'wb')
                            
                            # Open a binary file for reading
                            binary_file = open('data.bin', 'rb')
                            
                            # Open a binary file for both reading and writing
                            binary_file = open('data.bin', 'rb+')

10.3.3 close()

Just like with text files, it's important to close binary files when you're done with them.

Example 
# Open a binary file
                            binary_file = open('data.bin', 'wb')
                            
                            # Perform operations
                            # ...
                            
                            # Close the file
                            binary_file.close()

Best Practice: Use the with statement to automatically close binary files:

Using with statement 
with open('data.bin', 'wb') as binary_file:
                                # Perform operations
                                # ...
                            # File is automatically closed after the block

10.3.4 dump()

The dump() function from the pickle module serializes an object and writes it to a file object.

Syntax 
pickle.dump(obj, file, protocol=None)

Parameters:

  • obj: The Python object to be serialized
  • file: The file object where the serialized object will be written
  • protocol: The protocol version to use (optional)
Example 
import pickle
                            
                            # Data to be serialized
                            data = {
                                'name': 'John',
                                'age': 30,
                                'skills': ['Python', 'JavaScript', 'SQL']
                            }
                            
                            # Serialize and save to a binary file
                            with open('data.pickle', 'wb') as f:
                                pickle.dump(data, f)
                            
                            # Serialize multiple objects to the same file
                            items = [1, 2, 3, 4, 5]
                            more_data = {'a': 100, 'b': 200}
                            
                            with open('multiple_objects.pickle', 'wb') as f:
                                pickle.dump(items, f)
                                pickle.dump(more_data, f)

10.3.5 load()

The load() function deserializes a Python object from a file-like object.

Syntax 
obj = pickle.load(file)
Example 
import pickle
                            
                            # Load a serialized object from a file
                            with open('data.pickle', 'rb') as f:
                                loaded_data = pickle.load(f)
                                
                            print(loaded_data)  # Prints the original dictionary
                            
                            # Load multiple objects from the same file
                            with open('multiple_objects.pickle', 'rb') as f:
                                loaded_items = pickle.load(f)
                                loaded_more_data = pickle.load(f)
                                
                            print(loaded_items)      # Prints [1, 2, 3, 4, 5]
                            print(loaded_more_data)  # Prints {'a': 100, 'b': 200}

Note: When loading multiple objects, you must load them in the same order they were dumped.

10.3.6 Checking End of File

When reading binary files, it's important to know when you've reached the end of the file (EOF). The pickle module raises an EOFError exception when it tries to unpickle from an empty file or when all objects have been unpickled.

Example 
import pickle
                            
                            # Read all pickled objects from a file
                            objects = []
                            with open('multiple_objects.pickle', 'rb') as f:
                                while True:
                                    try:
                                        obj = pickle.load(f)
                                        objects.append(obj)
                                    except EOFError:
                                        break
                            
                            print(objects)  # Prints all unpickled objects

10.3.7 Deletion of records

Unlike databases, binary files don't support direct deletion of records. Instead, you need to read all the data, modify it in memory, and then write it back to the file.

Example: Deleting a record 
import pickle
                            
                            # Function to delete a record from a binary file
                            def delete_record(filename, key_to_delete):
                                # Read all records
                                records = []
                                with open(filename, 'rb') as f:
                                    while True:
                                        try:
                                            record = pickle.load(f)
                                            # Skip the record to be deleted
                                            if record['id'] != key_to_delete:
                                                records.append(record)
                                        except EOFError:
                                            break
                                
                                # Write back all records except the deleted one
                                with open(filename, 'wb') as f:
                                    for record in records:
                                        pickle.dump(record, f)
                            
                            # Example usage
                            students = [
                                {'id': 1, 'name': 'Alice', 'grade': 'A'},
                                {'id': 2, 'name': 'Bob', 'grade': 'B'},
                                {'id': 3, 'name': 'Charlie', 'grade': 'C'}
                            ]
                            
                            # Save records to a file
                            with open('students.pickle', 'wb') as f:
                                for student in students:
                                    pickle.dump(student, f)
                            
                            # Delete the record with id=2
                            delete_record('students.pickle', 2)
                            
                            # Verify deletion
                            with open('students.pickle', 'rb') as f:
                                while True:
                                    try:
                                        print(pickle.load(f))
                                    except EOFError:
                                        break

Note: For larger datasets, consider using a database system like SQLite, which provides more efficient ways to handle record management.

10.4 Object Oriented File Processing

Object-Oriented File Processing combines the power of OOP with file handling to create more maintainable and reusable code. This approach is particularly useful when working with complex data structures or when you need to perform multiple operations on the same file.

Benefits of OOP File Processing:

  • Encapsulation of file operations
  • Reusable code across multiple files
  • Better error handling and resource management
  • Abstraction of complex file operations
Example: TextFileManager Class 
class TextFileManager:
                                def __init__(self, filename, mode='r'):
                                    """Initialize the file manager with a filename and mode"""
                                    self.filename = filename
                                    self.mode = mode
                                    self.file = None
                                
                                def __enter__(self):
                                    """Context manager entry point"""
                                    self.file = open(self.filename, self.mode)
                                    return self
                                
                                def __exit__(self, exc_type, exc_val, exc_tb):
                                    """Context manager exit point"""
                                    if self.file:
                                        self.file.close()
                                
                                def read_all(self):
                                    """Read the entire file content"""
                                    if 'r' not in self.mode:
                                        raise ValueError("File not opened in read mode")
                                    return self.file.read()
                                
                                def read_lines(self):
                                    """Read all lines as a list"""
                                    if 'r' not in self.mode:
                                        raise ValueError("File not opened in read mode")
                                    return self.file.readlines()
                                
                                def write(self, content):
                                    """Write content to the file"""
                                    if 'w' not in self.mode and 'a' not in self.mode and '+' not in self.mode:
                                        raise ValueError("File not opened in write mode")
                                    self.file.write(content)
                                
                                def write_lines(self, lines):
                                    """Write multiple lines to the file"""
                                    if 'w' not in self.mode and 'a' not in self.mode and '+' not in self.mode:
                                        raise ValueError("File not opened in write mode")
                                    self.file.writelines(lines)
                            
                            # Example usage
                            with TextFileManager('example.txt', 'w') as file_manager:
                                file_manager.write("Hello, World!\n")
                                file_manager.write_lines(["Line 1\n", "Line 2\n", "Line 3\n"])
                            
                            with TextFileManager('example.txt', 'r') as file_manager:
                                content = file_manager.read_all()
                                print(content)
Example: BinaryFileManager Class using pickle 
import pickle
                            
                            class BinaryFileManager:
                                def __init__(self, filename, mode='rb'):
                                    """Initialize the binary file manager"""
                                    self.filename = filename
                                    self.mode = mode
                                    self.file = None
                                
                                def __enter__(self):
                                    """Context manager entry point"""
                                    self.file = open(self.filename, self.mode)
                                    return self
                                
                                def __exit__(self, exc_type, exc_val, exc_tb):
                                    """Context manager exit point"""
                                    if self.file:
                                        self.file.close()
                                
                                def save_object(self, obj):
                                    """Save a Python object to the binary file"""
                                    if 'b' not in self.mode or ('w' not in self.mode and 'a' not in self.mode and '+' not in self.mode):
                                        raise ValueError("File not opened in binary write mode")
                                    pickle.dump(obj, self.file)
                                
                                def load_object(self):
                                    """Load a Python object from the binary file"""
                                    if 'b' not in self.mode or 'r' not in self.mode:
                                        raise ValueError("File not opened in binary read mode")
                                    return pickle.load(self.file)
                                
                                def load_all_objects(self):
                                    """Load all objects from the binary file"""
                                    if 'b' not in self.mode or 'r' not in self.mode:
                                        raise ValueError("File not opened in binary read mode")
                                    
                                    objects = []
                                    try:
                                        while True:
                                            objects.append(self.load_object())
                                    except EOFError:
                                        pass
                                    
                                    return objects
                            
                            # Example usage
                            student = {
                                'name': 'John Doe',
                                'age': 21,
                                'courses': ['Math', 'Physics', 'Computer Science']
                            }
                            
                            # Save an object
                            with BinaryFileManager('student.pickle', 'wb') as bin_manager:
                                bin_manager.save_object(student)
                            
                            # Load an object
                            with BinaryFileManager('student.pickle', 'rb') as bin_manager:
                                loaded_student = bin_manager.load_object()
                                print(loaded_student)
Example: Student Record Management System 
import pickle
                            import os
                            
                            class Student:
                                def __init__(self, id, name, age, grade):
                                    self.id = id
                                    self.name = name
                                    self.age = age
                                    self.grade = grade
                                
                                def __str__(self):
                                    return f"Student(id={self.id}, name={self.name}, age={self.age}, grade={self.grade})"
                            
                            class StudentDatabase:
                                def __init__(self, filename):
                                    self.filename = filename
                                
                                def save_student(self, student):
                                    """Add a new student or update existing one"""
                                    students = self.get_all_students()
                                    
                                    # Update if student already exists, otherwise add new
                                    student_exists = False
                                    for i, s in enumerate(students):
                                        if s.id == student.id:
                                            students[i] = student
                                            student_exists = True
                                            break
                                    
                                    if not student_exists:
                                        students.append(student)
                                    
                                    # Save all students
                                    with open(self.filename, 'wb') as f:
                                        for s in students:
                                            pickle.dump(s, f)
                                
                                def get_student(self, student_id):
                                    """Get a student by ID"""
                                    students = self.get_all_students()
                                    for student in students:
                                        if student.id == student_id:
                                            return student
                                    return None
                                
                                def delete_student(self, student_id):
                                    """Delete a student by ID"""
                                    students = self.get_all_students()
                                    students = [s for s in students if s.id != student_id]
                                    
                                    # Save the updated list
                                    with open(self.filename, 'wb') as f:
                                        for student in students:
                                            pickle.dump(student, f)
                                
                                def get_all_students(self):
                                    """Get all students from the database"""
                                    students = []
                                    
                                    # Create the file if it doesn't exist
                                    if not os.path.exists(self.filename):
                                        open(self.filename, 'wb').close()
                                        return students
                                    
                                    # Read all students
                                    with open(self.filename, 'rb') as f:
                                        while True:
                                            try:
                                                student = pickle.load(f)
                                                students.append(student)
                                            except EOFError:
                                                break
                                    
                                    return students
                            
                            # Example usage
                            def main():
                                db = StudentDatabase('students.db')
                                
                                # Add students
                                db.save_student(Student(1, "Alice Smith", 20, "A"))
                                db.save_student(Student(2, "Bob Johnson", 22, "B"))
                                db.save_student(Student(3, "Charlie Brown", 21, "A-"))
                                
                                # Retrieve and display all students
                                print("All Students:")
                                for student in db.get_all_students():
                                    print(student)
                                
                                # Get a specific student
                                print("\nStudent with ID 2:")
                                student = db.get_student(2)
                                if student:
                                    print(student)
                                
                                # Update a student
                                updated_student = Student(2, "Bob Johnson", 23, "A")  # Changed grade and age
                                db.save_student(updated_student)
                                
                                # Delete a student
                                db.delete_student(3)
                                
                                # Display after changes
                                print("\nAfter updates:")
                                for student in db.get_all_students():
                                    print(student)
                            
                            if __name__ == "__main__":
                                main()

Key Advantages of OOP for File Handling

  • Encapsulation: File operations and associated data are bundled together
  • Inheritance: Create specialized file handlers by extending base classes
  • Abstraction: Hide complex file operations behind simple interfaces
  • Polymorphism: Use different file handlers interchangeably if they share the same interface
  • Resource Management: Context managers ensure proper resource cleanup
